Search Engine Optimisation


10 ways to improve your websites SEO.

  1. Summarize your website’s content as precise as possible. A brief description that is displayed when searching for your website should be relevant and quickly inform users about its content.
  2. Titles of webpages should be clear, relevant and under 65 characters. If a title is more than 65 it will be cut off.
  3. A sitemap or “navigational” page should be present to allow for easy navigation across your website.
  4. Implement meaningful URL links which can be understood by humans. For example www.website.com/index/23487583/324.html is not user-friendly, but www.website.com/index/seo-search-improvements.html is much easier to read.
  5. Page loading speed should be as fast as possible. According to QuickSprout (2019), 40% of users leave a website if the loading speed is longer than 3 seconds.
  6. Optimize your website’s content to improve the user experience. Multimedia such as images and videos are often what users want rather than just reading text.
  7. If your website includes multimedia, be sure to use alt tags to describe those elements.
  8. Make your website responsive for all form factors. This will allow for more traffic from many avenues.
  9. Include appropriate metadata when writing your html code.
  10. Ensure there is no “dead” code present in your website’s files.

Sources: (QuickSprout, 2019), (The Google Search Quality team, n.d.) & (Michigan Tech, n.d.)
